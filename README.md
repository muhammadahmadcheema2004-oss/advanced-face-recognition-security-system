# Advanced Face Recognition Security System

## Overview
This project is an AI-powered security system that performs real-time face recognition using FaceNet and MTCNN. The system identifies known individuals, detects unknown persons, and generates alerts when a wanted person is recognized.

The application combines deep learning, face embeddings, and computer vision to create a smart surveillance solution suitable for security and monitoring environments.
---

## Features
* Real-time face detection
* Face recognition using embeddings
* Known and unknown person classification
* Wanted person identification
* Audio warning alerts
* Profile information display
* GPU acceleration support
* Live webcam monitoring

---

## Technologies Used

---

## Workflow
1. Detect faces using MTCNN.
2. Generate facial embeddings using FaceNet.
3. Compare embeddings with stored identities.
4. Identify known individuals.
5. Detect unknown faces.
6. Trigger alerts for wanted persons.
7. Display information on screen.
